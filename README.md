# book
### Структура проекта
    Сборка проекта идет в каталоге "dist"
    Отладка в каталоге "app"

### Примечание
    npm версии 16.20.2
    nvm use v16.20.2

### команды
    запуск проекта для тестирования: gulp;
    Сборка проекта: gulp build;
    Очистка кеша: gulp clear;
    Положить все js скрипты в один и минимизировать его: gulp scripts
